# EID MUBARAK GREETING CARD 🌙

## Description
This project is a simple HTML-based Eid greeting card created to celebrate the joyful occasion of Eid al-Fitr.  
It focuses on a clean design and meaningful message to share festive wishes online.

## Features
- Simple and elegant design  
- Festive Eid greeting message  
- Clean layout and typography  
- Beginner-friendly HTML structure  

## Installation / Usage
1. Download or clone the repository  
2. Open the `index.html` file in your browser  
3. View and share the greeting card  

## Purpose
This project was created to practice basic HTML skills and build a real-life themed mini project.
